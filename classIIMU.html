IMU stands for Inertial Measurement Unit.

◆ getAccelerationValues()

virtual void IIMU::getAccelerationValues ( IMUData *  accelerationValues) const
pure virtual

Get last raw Accelerometer values as an IMUData struct containing values in x, y and z in digits.

Parameters
[in]accelerationValuesPointer to IMUData struct where the raw acceleration values in digits in x, y and z direction will be written into. The values can be converted into physical values in mm/s^2 via the multiplication with a sensitivity factor in mm/s^2/digit.
